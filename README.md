# Heart Disease Prediction using Machine Learning

## Project Overview
This project predicts whether a person has heart disease or not using medical data and Machine Learning (Logistic Regression).

## Dataset
The dataset contains patient health information such as age, chest pain type, blood pressure, cholesterol, heart rate, and other medical features.

## Workflow
- Data Cleaning (removed duplicates)
- Exploratory Data Analysis (EDA)
- Feature Selection
- Data Visualization
- Train-Test Split
- Feature Scaling
- Model Training (Logistic Regression)
- Model Evaluation

## Model Performance
- Accuracy: ~83%

## Key Insights
- Chest pain type is the strongest indicator of heart disease.
- Maximum heart rate and exercise-induced angina are important predictors.
- Age has a weak relationship with heart disease.
- Males show slightly higher cases of heart disease in this dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Conclusion
Logistic Regression provides a good baseline model for heart disease prediction with reasonable accuracy.
